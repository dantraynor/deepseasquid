⚠️ Links Not Working?

If clicking the slide links doesn't work, you need to serve the files via HTTP:

  1. Option 1: Use Python: python3 -m http.server 8000
  2. Option 2: Use Live Server extension in VS Code
  3. Option 3: Open files individually by dragging them into browser
  4. GitHub Pages: Deploy to GitHub Pages for permanent hosting
